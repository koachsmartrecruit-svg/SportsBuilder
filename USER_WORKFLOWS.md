## 1. Sports Academy Owner

**Profile**: Running a cricket/football/basketball academy with 50-200 students, needs to showcase programs and attract enrollments.

### Initial Setup Workflow

**Step 1: Account Creation**
- Visit the platform homepage
- Click "Sign Up"
- Enter: Name, Email, Password
- Verify email (check inbox for verification link)
- Login with credentials

**Step 2: Create Academy Website**
- Click "Create New Site" from dashboard
- Fill in basic information:
  - Site Name: "Elite Cricket Academy"
  - Slug: `elite-cricket` (URL will be: elite-cricket.sportsbuilder.com)
  - Template: Select "Academy Template"
  - Status: Keep as "Draft" for now

**Step 3: Customize Branding**
- Go to "Design" tab in admin dashboard
- Upload academy logo (PNG/JPG, recommended 200x200px)
- Set brand colors:
  - Primary Color: #1a472a (cricket green)
  - Secondary Color: #f39c12 (gold)
  - Accent Color: #e74c3c (red)
- Choose fonts:
  - Heading Font: Montserrat
  - Body Font: Open Sans

**Step 4: Add Content**
- Go to "Content" tab
- Hero Section:
  - Title: "Master Cricket with Professional Coaching"
  - Subtitle: "Join 200+ students learning from certified coaches"
  - CTA Text: "Book Free Trial"
  - CTA Link: "#contact"
- About Section:
  - Write 2-3 paragraphs about academy history, coaching philosophy
- Features (4 key offerings):
  - Feature 1: "Expert Coaches" - Icon: trophy
  - Feature 2: "Modern Facilities" - Icon: building
  - Feature 3: "Flexible Timings" - Icon: clock
  - Feature 4: "Performance Tracking" - Icon: chart
- Stats (4 achievements):
  - Stat 1: "200+ Students"
  - Stat 2: "15 Years Experience"
  - Stat 3: "5 Certified Coaches"
  - Stat 4: "95% Success Rate"
- Contact Information:
  - Email: info@elitecricket.com
  - Phone: +91-9876543210
  - WhatsApp: +91-9876543210
  - Address: "123 Sports Complex, Mumbai, Maharashtra 400001"
- Social Media:
  - Facebook: facebook.com/elitecricket
  - Instagram: @elitecricket
  - YouTube: youtube.com/elitecricket

**Step 5: Add Gallery Images**
- Click "Edit Site" from dashboard
- Scroll to Gallery section
- Upload 6-8 images:
  - Training sessions
  - Coaching moments
  - Facility photos
  - Student achievements
  - Team photos
- Add captions to each image
- Save changes

**Step 6: SEO Optimization**
- Go to "SEO" tab in admin
- Meta Title: "Elite Cricket Academy Mumbai | Professional Cricket Coaching"
- Meta Description: "Join Mumbai's premier cricket academy. Expert coaches, modern facilities, flexible timings. Book your free trial today!"
- Meta Keywords: "cricket academy mumbai, cricket coaching, professional cricket training"
- Upload OG Image (social media preview image)

**Step 7: Preview & Test**
- Click "Preview" button
- Test on mobile and desktop
- Check all links work
- Verify contact form submission
- Test WhatsApp click-to-chat

**Step 8: Publish**
- Go to "Settings" tab
- Toggle "Published" to ON
- Click "Save Settings"
- Copy public link: `https://elite-cricket.sportsbuilder.com`
- Share on social media

### Daily Operations Workflow

**Morning Routine (10 minutes)**
1. Login to dashboard
2. Check "Overview" tab for new submissions
3. Review contact form entries
4. Mark urgent inquiries
5. Respond to inquiries via email/phone

**Weekly Content Updates (30 minutes)**
1. Add new gallery images from recent sessions
2. Update stats if milestones reached
3. Post achievement updates in about section
4. Check analytics for visitor trends

**Monthly Maintenance (1 hour)**
1. Review and update program pricing
2. Add seasonal camp information
3. Update coach bios if staff changes
4. Refresh testimonials
5. Check broken links
6. Update meta description for SEO

## Troubleshooting Common Issues

### Issue: Not receiving contact form submissions
**Solution:**
1. Check spam folder
2. Verify email in contact settings
3. Test form submission yourself
4. Check "Submissions" tab in admin

### Issue: Images not displaying
**Solution:**
1. Check image file size (compress if >2MB)
2. Use supported formats (JPG, PNG)
3. Re-upload image
4. Clear browser cache

### Issue: Site not showing in search
**Solution:**
1. Ensure site is published
2. Update SEO meta tags
3. Share on social media for backlinks
4. Give it time (2-4 weeks for indexing)

### Issue: Slow loading
**Solution:**
1. Compress images before upload
2. Limit gallery to 10-15 images
3. Remove unused content
4. Check internet connection

### Issue: Can't edit site
**Solution:**
1. Ensure you're logged in
2. Check if you're the site owner
3. Try different browser
4. Clear cache and cookies
